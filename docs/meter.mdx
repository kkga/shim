## Examples

### Custom range and labels

Use the `label` and `description` props to provide additional context. Use
`minValue` and `maxValue` to set the range of the meter, see more in the
[API documentation](https://react-spectrum.adobe.com/react-aria/Meter.html#props).

<Demo
  className="items-stretch"
  demo={<MeterDemo.Custom />}
  code={demos['custom']}
/>

### Bar color

Use the `color` prop to change the color of the meter bar without modifying the
component source. The color can be a CSS color string or a function that takes
the current value and returns a color string.

  className="items-stretch"
