### States

Use `isSelected`, `isDisabled` and `isIndeterminate` props to set the state of
the checkbox.

<Demo demo={<CheckboxDemo.States />} code={demos["states"]} />

### Group

Use the `CheckboxGroup` component to group checkboxes together with an
accessible label. The `label` prop is required to provide an accessible name for
the group.

<Demo demo={<CheckboxDemo.Group />} code={demos["group"]} />

### Size

Use `size` prop to set the size of the group or individual checkboxes.

<Demo demo={<CheckboxDemo.Size />} code={demos["size"]} />

### Variant

Use `variant` prop to set the variant of the group or individual checkboxes.
