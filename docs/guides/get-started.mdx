## Overview

These components are built on top of the
[React Aria](https://react-spectrum.adobe.com/react-aria/index.html) library,
which provides a set of hooks and components for building accessible UI
components.

The components are written in TypeScript and utilize Tailwind CSS v4 and [cva]()
for styling.

## Install instructions

### React Aria Components

Install the `react-aria-components` package using your package manager of
choice.

```sh
pnpm add react-aria-components
```

### Tailwind CSS v4

Install the `tailwindcss` package using your package manager of choice.

```sh
pnpm add tailwindcss@next
```

### cva

Install the `cva` package using your package manager of choice.

```sh
pnpm add cva@beta
```
