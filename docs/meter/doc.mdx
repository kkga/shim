### Custom range and labels

Use the _label_ and _description_ props to provide additional context. Use
_minValue_ and _maxValue_ to set the range of the meter, see more in the
[API documentation](https://react-spectrum.adobe.com/react-aria/Meter.html#props).

  code={demos["custom"]}
/>

### Bar color

Use the _color_ prop to set a custom color for the meter bar. The color can be a
CSS color string or a function that takes the current value and returns a color
string.
