
<Demo

  demo={<MenuDemo.Content />}
  code={demos.content}
>
  Menu follows the [Collections
  API](https://react-spectrum.adobe.com/react-aria/collections.html) and
  accepts both static and dynamic items.

To use dynamic items, pass an `items` prop to the Menu component and a render
function as children. The render function will be called with each item in the
`items` iterable and should return a React element.

</Demo>

<Demo title="Selection" demo={<MenuDemo.Selection />} code={demos.selection}>
  Menu supports single selection, multiple selection, and no selection. 
  
  To enable selection, pass a `selectionMode` prop to the Menu component. The
  default is `none`.

</Demo>

To group items in a menu, use the `<MenuSection>` component. It accepts a

<Demo
  title="Submenu"

  demo={<MenuDemo.Submenu />}
  code={demos.submenu}
>
To create a submenu, use the `<SubmenuTrigger>` component. It accepts two
children: a trigger element and a submenu element. The submenu element should be
a `<Menu>` component.
</Demo>
