## What is Shim?

Copy-paste UI components built on top of [React Aria Components](https://react-spectrum.adobe.com/react-aria/index.html). You own the code.

## Shim setup

<Steps>
  <Step>
    ### Configure path alias

    Shim components import files via the `@/` alias. Configure the path in `tsconfig.json` (or `jsconfig.json`) so those imports resolve:

    <CodeBlock
      code={[
        {
          title: "tsconfig.json",
          content: `{
    "compilerOptions": {
      "baseUrl": ".",
      "paths": {
        "@/*": ["./*"]
      }
    }
}`,
        },
      ]}
    />

  </Step>
  <Step>
    ### Initialize Shim

    Run the `init` command to initialize Shim. 
    This command:

    - Creates `shim.config.json` with your chosen paths
    - Downloads the required utility files (`style.ts`, `theme.tsx`)
    - Fetches the theme CSS file (`theme.css`)


    <CodeBlock code={[{ title: "Terminal", content: "pnpm dlx @kkga/shim init", }]} />

  </Step>
</Steps>


You can customise the paths during setup:

```bash
pnpm dlx @kkga/shim init --components-path src/ui --utils-path src/lib --css-path src/styles
```

Sample configuration:

<CodeBlock
  code={[
    {
      title: "shim.config.json",
      content: `{
"componentsPath": "src/ui",
"utilsPath": "src/lib",
"cssPath": "src/styles"
}`,
    },
  ]}
/>

## Add components

### Using the CLI

```bash
pnpm dlx @kkga/shim add button dialog
```

The CLI resolves dependencies, rewrites imports, and installs into the paths defined in `shim.config.json`.

### Manual copy-paste

1. Browse the component source in the [Shim repository](https://github.com/kkga/shim/tree/master/shim-ui)
2. Copy the component `.tsx` file into your project
3. Copy required utility files (`lib/style.ts`, `lib/theme.tsx`) and `css/theme.css`
4. Update import paths (or set up a path alias) so component imports continue to resolve. Aliases keep your project structure flexible and match what the CLI generates.
