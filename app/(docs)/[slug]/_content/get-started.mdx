Shim is a pre-styled collection of copy-paste UI components built on top of [React Aria Components](https://react-spectrum.adobe.com/react-aria/index.html). 

## Install the prerequisites

Shim components depend on several libraries. Use the commands below to install them.

<Step
  title="React Aria Components"
  code="pnpm add react-aria-components" 
  description={
    <p>
      Core unstyled primitives used by Shim components.<br/>
      Read the docs at <a href="https://react-spectrum.adobe.com/react-aria/components.html">react-spectrum.adobe.com/react-aria/components</a>.
    </p>
  }
/>
